Cấu Hình Environment

Thiết lập các biến môi trường cần thiết cho dự án.

File .env cần có:
# Google Sheets Configuration
GOOGLE_CREDENTIALS_PATH=./google-credentials.json
GOOGLE_SHEET_ID=your_sheet_id_here
GOOGLE_SHEET_NAME=OneAutomationData

# Database
MONGODB_URI=mongodb://localhost:27017/oneautomation
DB_NAME=oneautomation

# JWT Authentication
JWT_SECRET=your-super-secret-jwt-key-here
JWT_EXPIRES_IN=7d

# Server Configuration
PORT=3001
NODE_ENV=development

# One.com API (sẽ cập nhật sau)
ONE_API_KEY=your_one_api_key
ONE_API_SECRET=your_one_api_secret
ONE_BASE_URL=https://api.one.com

# Email Service
EMAIL_SERVICE=gmail
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password

Kiểm Tra Kết Nối

Test kết nối Google Sheets API và đảm bảo mọi thứ hoạt động đúng.

Script kiểm tra kết nối:
// test-sheets-connection.js
const { google } = require('googleapis');
const fs = require('fs');
require('dotenv').config();

async function testSheetsConnection() {
    try {
        // Load credentials
        const credentials = JSON.parse(fs.readFileSync(process.env.GOOGLE_CREDENTIALS_PATH));
        
        // Initialize auth
        const auth = new google.auth.GoogleAuth({
            credentials,
            scopes: ['https://www.googleapis.com/auth/spreadsheets']
        });

        // Create sheets service
        const sheets = google.sheets({ version: 'v4', auth });
        
        // Test read operation
        const response = await sheets.spreadsheets.values.get({
            spreadsheetId: process.env.GOOGLE_SHEET_ID,
            range: 'Dashboard!A1:F10'
        });

        console.log('✅ Connection successful!');
        console.log('Rows returned:', response.data.values?.length || 0);
        console.log('Sample data:', response.data.values?.[0] || 'No data');
        
        return true;
    } catch (error) {
        console.error('❌ Connection failed:', error.message);
        return false;
    }
}

testSheetsConnection();

Build Production

Chuẩn bị ứng dụng cho môi trường production.

Frontend Build:

npm run build

# Verify build folder created

Backend Setup:
# Install production dependencies
cd backend
npm ci --only=production

# Set production environment
export NODE_ENV=production
